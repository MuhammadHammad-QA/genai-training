## Prompt Engineering

Created 10 prompts with increasing difficulty:
- Simple extraction
- Classification
- Nested JSON
- Reasoning
- Complex structured outputs


## Observations

- Summarize-then-answer reduced token usage significantly.
- JSON schema instructions improved response consistency.
- Simple prompts produced accurate structured outputs.
- Complex prompts required clearer instructions and constraints.
- JSON validation helped verify model output format.


## Conclusion

Summarization improves efficiency for large documents, while structured prompts and JSON schemas make LLM responses more reliable and easier to process.
